Import Data

Data can be imported as Text Figure 1 or as Binary Figure 2.


When Data is imported as text the Encoded option needs to be left unchecked.


When Data is imported as Binary the Encoded option needs to be checked. Binary data is imported as Base64 and the Encoded option informs the program to decode the base64 data back to it original format before writing to disk.
