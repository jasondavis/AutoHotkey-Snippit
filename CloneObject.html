Clone Object

Clone Object will create a duplicate of the current Hotstring/ Hotkey / Inlcude / Data


Clone Object will clone the current active Tab.


Note:

Cloning a Hotkey will ill set it to disabled. After Making the necessary changes don't for get to enable the clone.

Cloning a Hotstring ill set it to disabled. After Making the necessary changes don't for get to enable the clone.

Cloning an Include will ill set it to disabled. After Making the necessary changes don't for get to enable the clone.

Cloning a Data Item will rename the Name and File Name. Also the Overwrite option will be set to false. Adjust the options before applying changes.
