Include Edit

Editing an Include from the Hotlist  can be done by Selecting a Plugin that contain the Include to edit and then clicking on the Edit Selected Plugin Option as shown in figure 1 or by right clicking on an include in the list and choosing edit as show in figure 2. This will take you to the Includes edit screen as shown in figure 4.


Note: If edit button is clicked with no item selected in the list then your will start on the Plugin window on the Main tab as shown in figure 3. Click on the Includes tab and then navigate to the Include you want to edit.
