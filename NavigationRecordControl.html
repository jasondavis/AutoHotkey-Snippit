Navigation Record Control

The Navigation record control is used on various forms.


Go to first. Button will navigate to the first record in the set.


Go to previous. Button will navigate to previous recored in the set.


Go to input. Input the value of the record to jump to.


Go to Next. Button will navigate to next record in the set.


Go to Last. Button will navigate to last record in the set.


Add New. Button will add a new record to the set.


Delete. Button will delete the current record from the set.
