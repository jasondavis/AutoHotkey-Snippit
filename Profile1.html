Profile

A Profile is a group of settings that determine how the Plugins can interact with the system.
The Profile mainly consist of Global option, End Chars and Windows.

  1. Global options affects if the profile is system wide or localized to selected windows.
  2. End Chars affect how any Hotstring's are triggered.
  3. Windows decide what window the profile is applied to ( if not Global) or Not applied to ( if Global).
